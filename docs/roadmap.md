# Roadmap

## v1.1

- markdown-friendly audit report export;
- richer duplicate SKU grouping for operator review;
- optional category-level audit thresholds.

## v1.2

- alternative export formats besides the current Excel catalog;
- improved operational diagnostics around skipped input files.

## v1.3

- XML or CommerceML-oriented export exploration;
- stronger release-oriented packaging metadata.

## Принцип развития

После `v1.0.0` проект уже считается архитектурно зрелым.

Поэтому развитие должно сохранять текущий стиль:

- сначала локальное решение;
- затем только необходимое расширение;
- без усложнения ради абстракции;
- без переноса бизнес-логики между слоями без явной причины.
